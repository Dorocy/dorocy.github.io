생산 데이터 기반 실시간 데이터 연동을 통한 DPP 자동 생성 서비스

 Github
목적 : 공장의 기존 시스템을 변경없이 데이터ETL을 통한 DPP 생성 자동화
문제점 :
   - 정적 데이터와 동적 데이터 분산된 현장 시스템
   - 수동 데이터 확인 및 추출로 인한 DPP 생성 어려움
기술 스택 : Typescript, Node-red, MS-SQL, MongoDB
주요 기능 : ETL 파이프라인 구축, AAS 정보모델 자동 생성, DPP 변환 및 DB 저장
성과 :
    - 논문 1건, 특허 1건, SW 등록 1건
    - 실시간 연동을 통한 DPP 생성 자동화 및 높은 효율성
    - 기존 생산 시스템과의 높은 호환 및 연동성

AAS의 의미적 상호운용성을 위한 지식그래프기반 SemanticID 발견 연구

 Github
목적 : 수동적 SemanticID 검색 및 입력으로 인한 한계 극복을 위한 자동 의마 매칭
문제점 :
   - 수동으로 SemanticID를 입력하는 것은 많은 시간소요
   - 수동 입력으로 인한 작성자 직관적 판단으로 일부 데이터 제외 및 누락 우려
   - 새로운 SemanticID 정보 반영이 어려움
기술 스택 : KnowledgeGraph, Python, Similarity, LLM, ETL
주요 기능 : 외부 지식 활용(Wikidata KG), 지식그래프 생성, 그래프 유사도 비교
성과 :
   - 석사 논문 1건,
   - 의미 자동 매칭을 통한 시간 및 효율성 향상
    - LLM 모델 연계를 통한 기술 고도화 연구 진행 중
